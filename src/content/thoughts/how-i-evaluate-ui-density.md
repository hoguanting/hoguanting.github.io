Density is not just about fitting more items on a page. It is about making information feel calm at a glance. Tight spacing can improve scanning, but only if typography, contrast, and alignment are doing the structural work.

When a layout feels noisy, the problem is often not that it is too compact. The problem is that too many elements are competing at the same visual volume.
